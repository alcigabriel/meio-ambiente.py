# meio-ambiente.py

O projeto foi desenvolvido com o objetivo de monitorar o consumo de recursos naturais, como água, energia e reciclagem. A proposta busca conscientizar o usuário sobre práticas sustentáveis por meio da análise dos dados inseridos.

O sistema foi implementado em Python, utilizando lógica de programação com estruturas condicionais, laços de repetição e funções. Os dados são inseridos pelo usuário, processados pelo sistema e armazenados em um banco de dados SQLite, permitindo consultas e análises posteriores.

Como integração com outras disciplinas, eu utilizei banco de dados para garantir a persistência das informações. O código foi organizado de forma modular, facilitando a leitura e manutenção.

O sistema permite registrar dados, visualizar histórico, calcular médias e gerar alertas com base no consumo.
